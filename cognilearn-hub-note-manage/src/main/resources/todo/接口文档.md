一、核心后端技术栈


| 技术                | 作用说明                                  |
| ------------------- | ----------------------------------------- |
| **Spring Boot 3.2** | 快速构建后端应用，依赖注入，模块化支持    |
| **Spring Data JPA** | ORM 框架，实现实体和数据库表映射          |
| **Spring Security** | 权限认证和访问控制                        |
|                     |                                           |
| **PostgreSQL 15+**  | 主数据库，支持 JSONB、全文搜索、ltree 等  |
| **Redis**           | 缓存热数据，如频繁访问的笔记、权限信息等  |
| **MinIO**           | 对象存储，支持附件、图片等文件的存储      |
| **PGVector**        | PostgreSQL 插件，用于向量搜索（语义检索） |

---

二、AI集成技术栈


| 技术                | 作用说明                                   |
| ------------------- | ------------------------------------------ |
| **Spring AI**       | Spring 生态下的统一 AI 接口调用框架        |
| **HuggingFace API** | 开源模型接入，如文本摘要、关键词提取等     |
| **OpenAI SDK**      | GPT 系列接入，支持高级文本生成、语法纠错等 |
