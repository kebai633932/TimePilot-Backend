# 🕒 TimePilot-Backend

> Backend service of **TimePilot**, a lightweight and efficient time management & learning plan system.

---

### 🚀 项目简介

`TimePilot-Backend` 是 **TimePilot 时间管理系统** 的后端核心服务，  
负责提供账户认证、任务计划、事件管理等主要业务功能的 API 支撑。

项目采用 **Spring Boot + postgres + Redis + JWT** 架构设计，  
为移动端（TimePilot-App）与 Web 前端提供统一的认证与数据服务接口。

#### 🎯 核心目标
- 📅 支撑用户的日程安排与习惯养成  
- 🔒 构建安全、可扩展的认证与授权体系  
- ⚡ 提供高性能的异步与定时任务处理机制
